# av1-parser-gui

A browser based AV1 bitstream parser and visualizer.
Live App: https://mdakram.com/av1-parser-gui/

### Preview
![](docs/output.gif)

## Usage

To install and start, follow these steps:

```bash
# Get the code
git clone https://github.com/yourusername/av1-parser-gui.git
cd av1-parser-gui
# Build
npm install
# Run
npm start
```
